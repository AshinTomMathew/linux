top Command

top is used to display the real-time process activity in Linux. It shows system resource usage like CPU, memory, and processes running on the system.

Syntax:

top [options]

Options:

  • -d : Set the delay between updates.
  • -n : Set the number of iterations before quitting.
  • -u : Show processes for a specific user.
  • -p : Monitor a specific process ID.

kill Command

kill sends signals to processes, usually to terminate them. The default signal is SIGTERM (signal 15), but other signals can be specified.

Syntax:

kill [options]

Options:

  • -9: Forcefully terminate the process (SIGKILL).
  • -l: List all available signals.
  • -s : Specify a signal.

ifconfig Command

ifconfig is used to configure and display network interfaces in Unix-based operating systems.

Syntax:

ifconfig [interface] [options]

Options:

  • interface: The network interface (e.g., eth0, wlan0).
  • up: Bring the interface up.
  • down: Bring the interface down.
  • inet
    : Set the IP address.

ping Command

ping is used to check network connectivity by sending ICMP Echo Request packets to a remote host.

Syntax:

ping [options]

Options:

  • -c : Send a specific number of ping requests.
  • -i : Set the interval between packets.
  • -t : Set the Time To Live (TTL).

Example:

ssh Command

ssh is used to securely connect to remote machines over a network using encrypted communication.

Syntax:

ssh [options] @

Options:

  • -p : Specify a custom SSH port.
  • -i : Use a specific private key file for authentication.
  • -v: Enable verbose output.

Example:

scp Command

scp is used to securely transfer files between local and remote systems using SSH encryption.

Syntax:

scp [options]

Options:

  • -r: Recursively copy directories.
  • -P : Specify a port number.
  • -i : Use a private key for authentication.

Example:

curl Command

curl is used to transfer data from or to a server using various protocols, such as HTTP and FTP.

Syntax:

curl [options]

Options:

  • -O: Download a file from a URL.
  • -L: Follow redirects.
  • -u : Authenticate using a username and password.

Example:

diff Command

diff is used to compare the contents of two files line by line.

Syntax:

diff [options]

Options:

  • -u: Display output in a unified format.
  • -c: Display output with context.

Example:

head Command

head is used to display the first few lines of a file.

Syntax:

head [options]

Options:

  • -n : Specify the number of lines to display (default is 10).

Example:

tail Command

tail is used to display the last few lines of a file.

Syntax:

tail [options]

Options:

  • -n : Specify the number of lines to display (default is 10).
  • -f: Continuously output new lines as they are added to the file.
